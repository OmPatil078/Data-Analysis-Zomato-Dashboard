# Zomato Restaurant Analytics Dashboard (MySQL & Tableau)

## Project Objective

The objective of this project is to analyze restaurant data from Zomato and develop an interactive dashboard that provides insights into restaurant distribution, ratings, pricing, cuisines, online delivery, and table booking trends. This helps identify customer preferences and support data-driven business decisions.

## Dataset Used

- <a herf="https://github.com/OmPatil078/Data-Analysis-Zomato-Dashboard/blob/main/Country%20code%20clean.xlsx">Country Code </a>
- <a herf="https://github.com/OmPatil078/Data-Analysis-Zomato-Dashboard/blob/main/Zomato%20Clean.xlsx">Zomato Data </a>

## Questions (KPIs)

* Find the number of restaurants based on Country and City.
* Analyze restaurant openings by Year, Quarter, and Month.
* Count restaurants based on Average Ratings.
* Create price buckets and analyze restaurant distribution.
* Calculate the percentage of restaurants offering Table Booking.
* Calculate the percentage of restaurants providing Online Delivery.
* Identify top cuisines based on restaurant count.
* Analyze city-wise and country-wise restaurant performance.
* Evaluate restaurant ratings and pricing trends.

## Process

* Imported and cleaned Zomato and Country datasets.
* Created a Country Mapping Table for geographical analysis.
* Built a Calendar Table using DateKey and generated Year, Month, Quarter, Financial Month, and Financial Quarter fields.
* Wrote SQL queries to perform restaurant, rating, pricing, and delivery analysis.
* Connected MySQL data to Tableau for visualization.
* Developed interactive dashboards using filters and dynamic visualizations.
